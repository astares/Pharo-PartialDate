# Pharo-PartialDate
**PartialDate** implementation for [Pharo](http://www.pharo.org) to deal with (yet) incomplete dates

# Quick Start

## Installation via Script
```Smalltalk
Metacello new 
	repository: 'github://astares/Pharo-PartialDate:main/src';
	baseline: 'PartialDate';
	load
```
